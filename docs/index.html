How It Works

1

Submit via GitHub Issues

Use structured issue forms to submit challenges, ideas, or final solutions. Everything is version-controlled and public.

2

AI Evaluation Pipeline

GitHub Actions automatically triggers evaluation via the GitHub Models API (GPT-4o). Scores are posted as issue comments within minutes.

3

Live Leaderboard

The leaderboard updates automatically after every evaluation. Track rankings across 5 dimensions: Innovation, Technical Quality, Documentation, Feasibility, and Impact.

4

Human + AI Winner Selection

AI scores inform human judges who finalize the rankings. Winners are announced via GitHub Releases.
